# icaveri
## Launcher software  for iverilog4win.
---
For Verilog Circuit Designer.

[ Using other software ]
- Icarus Verilog for Windows.
- GTKWave for windows (included in Icarus Verilog)
- VisualStudio Code  
  or Sakura-Editor

[ Function ]
- Test-set is selectable.
- 1 bottun to simulation.
